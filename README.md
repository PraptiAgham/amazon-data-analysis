# Amazon Product Sales Analysis
**Data Analyst Portfolio Project — Prapti Agham**

## 📌 Overview
Exploratory Data Analysis on Amazon Sales Dataset (Kaggle) — 
1,465 products across 9 categories.

## 🔍 Key Findings
- Products with 40–60% discount have the highest avg ratings (4.2+)
- Electronics accounts for 58% of all listed products
- 82% of products are rated between 4.0–5.0 stars

## 🛠️ Tools Used
Python | Pandas | SQL | Matplotlib | Power BI | Excel | Chart.js
